Prework Study Guide

HTML

  • The head element repsents information that's used for the webpage to render. Invisible to user
  • The body element is visible to the user and this is the content of the webpage.
  • h1, h2, h3, ... are the headers on the webpage in the order of largest to smallest.
  • The main element is the dominant content of the body element that's the central topic of the webpage.

CSS

  • You can link a CSS file to your HTML file by the link tag included in the head section
  • class attributes enables you to choose while elements will be assigned the CSS rule
  • The class is indicated for a section element with the class attribute
  • Use .class-name in the CSS to rule to assign this CSS rule to all class-name elements in the HTML file

Git

  • git status: Checks what branch I'm currently in!
  • git checkout main: Goes to the main branch.
  • git checkout -b feature-branch-name: Used to create a feature branch in Git to develop a feature in isolation.
